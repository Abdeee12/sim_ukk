# README  SIMM (Sistem Informasi Magang)

Ringkasan singkat proyek SIMM: aplikasi Laravel untuk mengelola laporan harian siswa magang, penugasan mentor, dan administrasi peserta.

- Peran: `admin`, `mentor`, `siswa`.
- Fitur utama: laporan harian (create/edit/approve/reject), manajemen siswa & mentor (CRUD + penugasan), upload bukti dan tanda tangan mentor.
- Lokasi kode utama: `app/Models`, `app/Http/Controllers`, `resources/views`.

Dokumentasi lebih lengkap ada pada file lain di folder ini: `QUICKSTART.md`, `ROUTES.md`, `ERD.md`.

Jika Anda ingin saya menambahkan petunjuk lain (contoh: export, setup database, atau skrip tambahan), beri tahu saya.
